# Responsiv Landing Page

Bu layihə HTML5, CSS3 və Vanilla JavaScript istifadə edilərək hazırlanmış responsiv (Responsive) Landing Page nümunəsidir. Layihənin məqsədi semantik HTML, mobile-first yanaşması, responsiv dizayn, JavaScript interaktivliyi, forma validasiyası və əlçatanlıq (Accessibility) prinsiplərini tətbiq etməkdir.

## İstifadə olunan texnologiyalar

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

## Layihənin xüsusiyyətləri

### Semantik HTML

* `header`
* `nav`
* `main`
* `section`
* `article`
* `footer`
* Düzgün heading iyerarxiyası (`h1 → h2 → h3`)

### Responsiv Dizayn

* Mobile First yanaşması
* Flexbox
* CSS Grid
* 3 Breakpoint:

  * Mobil (0–767px)
  * Planşet (768–1023px)
  * Desktop (1024px və yuxarı)

### JavaScript Funksionallığı

* Mobil menyunun açılıb-bağlanması (Menu Toggle)
* Smooth Scroll

### Əlaqə Formu

* Client-side validasiya
* Boş sahələrin yoxlanılması
* Email Regex yoxlanışı
* Xəta mesajlarının göstərilməsi
* Uğurlu göndərilmə mesajı

### Accessibility (Əlçatanlıq)

* `alt` atributları
* `aria-label`
* `aria-expanded`
* `aria-controls`
* Klaviatura ilə naviqasiya (`:focus-visible`)
* Yüksək rəng kontrastı
* `label` və form elementlərinin əlaqələndirilməsi

1. Repository-ni klonlayın:

```bash

```

2. Layihə qovluğuna keçin:

```bash

```
